ngx-datawindow

借鉴 PowerBuilder DataWindow 设计哲学的 Angular 表格组件 — DataStore 数据引擎、虚拟计算列、列级变更跟踪、离线持久化

🚀 快速开始 📦 GitHub

核心特性

企业级表格组件应有的能力

🗄️

DataStore 数据引擎

三缓冲区架构（Main/Filtered/Deleted），行级状态管理（new/normal/modified/deleted），完整的 CRUD 操作和事件链

🔢

虚拟计算列

声明式 formula 定义，支持四则运算和条件表达式，修改源列自动联动重算，虚拟列不存储只计算

📝

列级变更跟踪

精确到字段级别的 oldValue/newValue 追踪，FieldChangeHistory 完整记录，支持差异更新生成

📴

离线优先架构

IndexedDB 持久化，Online/Offline 自动切换，乐观锁冲突检测（server_wins/client_wins/manual 三策略）

📊

聚合统计

支持 sum/avg/count/min/max/distinct 等聚合类型，按字段分组聚合，列头聚合标签显示

🎨

Angular Material 集成

基于 MatTable 二次封装，完整的工具栏/分页/排序/过滤，内联编辑 + 行选择

快速开始

三步完成集成

安装

npm install ngx-datawindow

导入模块

import { DataTableModule } from 'ngx-datawindow'; @NgModule({ imports: [DataTableModule] }) export class AppModule {}

使用组件

<!-- 最简使用 --> <ngx-datawindow [columns]="columns" [data]="data" [tableConfig]="config"> </ngx-datawindow>
